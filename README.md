**📊 PhonePe Data Visualization Project**

📌 Overview
This project focuses on analyzing PhonePe digital transaction data to understand payment trends, user growth, and regional performance across India. The goal is to transform raw data into meaningful insights using data cleaning, analysis, and interactive dashboards.

**🎯 Objectives**
Analyze digital payment trends over time
Identify top-performing states and districts
Understand user behavior and transaction patterns
Build an interactive dashboard for better decision-making
🛠️ Tools & Technologies
Power BI
Excel / CSV
Python (optional)
TopoJSON

**🔄 Data Processing Steps**
Imported raw transaction datasets
Cleaned and standardized column names
Handled missing and inconsistent values
Matched district names with map (TopoJSON)
Created calculated columns and measures
Built relationships between datasets

**📈 Key Insights**
Growth of digital transactions over years
State-wise and district-wise performance comparison
Seasonal trends in transaction volume
User engagement patterns across regions

**📊 Dashboard Features**
Transaction trends over time (Year/Quarter)
State-wise performance map
District-level analysis
Interactive filters and slicers
KPI cards for quick insights
